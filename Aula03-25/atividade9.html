Crie um script em JavaScript que resolva o clássico problema do FizzBuzz e exiba o resultado dentro de uma div na página utilizando innerHTML. Regras do FizzBuzz: Para números de 1 a 100: Se o número for múltiplo de 3, exiba "Fizz". Se o número for múltiplo de 5, exiba "Buzz". Se o número for múltiplo de 3 e 5, exiba "FizzBuzz". Caso contrário, exiba o próprio número.
